# letterboxd-film-picker# Générateur de films Letterboxd

Application Streamlit pour tirer un film aléatoire depuis votre watchlist Letterboxd exportée en CSV.

## Utilisation

1. Exportez votre watchlist Letterboxd au format CSV.  
2. Ouvrez l’application.  
3. Importez le CSV avec le bouton “Choisir un fichier”.  
4. Cliquez sur “Nouveau film” pour générer un film aléatoire.  
5. Cliquez sur le bouton pour aller sur Letterboxd.
